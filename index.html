How It Works

From setup to
full operation

Get your hospital running in minutes. MedCore connects directly to your backend REST API with zero configuration.

01
Start the backend
Launch your Flask server on port 5000. MedCore auto-connects and pulls live data on every page load.
02
Add departments & doctors
Set up your hospital structure — departments first, then assign doctors with their specializations and status.
03
Register patients
Add patient records with demographics. The system tracks gender distribution and age analytics automatically.
04
Schedule & bill
Create appointments with the cascading dept→doctor selector. Generate bills tied to appointments and print receipts.
